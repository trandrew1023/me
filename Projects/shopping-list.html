Shopping List

A class mini-project to create an app meeting criteria.

This is a simple shopping list Android app created to learn how to build an Android app from the ground up. There was also a focus on using the MVC pattern and specific tools and libraries.

Functionality:

  • Users can create a new shopping list
  • Users can name and rename a shopping list
  • Users can add items to a shopping list
  • Users can remove items from a shopping list
  • Users can navigate between shopping lists
  • Users can save a shopping list, or shopping lists are automatically saved, and saved listspersist across multiple runs of the app
  • Users can delete a shopping list
  • Users can duplicate a shopping list
  • The app supports both portrait and landscape mode, and the state of the view and datais not lost when switching between portrait and landscape mode
  • The app works well on a variety of screen sizes
